# Chaise Hub

A responsive Next.js dashboard flow that lets freelancers book a Chaise Hub workspace and create direct client contracts with escrow protection.

Live: https://chaise-hub.vercel.app

## Included

- Project name, details, timeline, value, and editable milestones
- Hub centre, work date, and working hours
- Estimated 5% Chaise Hub service fee with explicit consent
- Unique access code and downloadable QR pass
- Contract menu item after Hub
- Direct contract creation with fixed-price, hourly, and milestone terms
- Optional Hub workspace fee on contracts
- Client review, comments/change request, account, and demo escrow funding flow
- Responsive desktop and mobile dashboard layouts

## Run locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
